DF 1.7 Release Notes

New Features

Traceability

A dedicated app helps achieve complete production traceability records

The new Location Manager App enables operations personnel to track, validate and register inventory per location. The app will validate that materials adhere to preset requirements and alert against rule violations. Together with the WEB application, the Location Manager App provides an out-of-box solution for manufacturers seeking to adhere to rigorous industry standards for maintaining online traceability records for product and batch genealogy.

Contact support@emerald.co.il for details and pricing.

Setup Granularity

System Administrators now control how working setup times affect machine availability

We've added the option to log setup times as 2 different categories: working and non-working depending on machine activity. This feature is optional and can be activated independently for each machine. To date, your Live Production Floor indicates whether machines in setup are working or stopped, and all setup time is logged as downtime.

Activate this new feature to count working setup times toward production time and non-working setup times toward downtime. Activation of this option will result in adjustments to performance indicators, including Machine Availability, OEE and P.E.
Learn more.

Machine Idling

Easily distinguish between machine stops and planned idling in real time

Inactivity Calendars now update stop reasons right from the outset as soon as a machine stop is detected. Use the calendars to plan idle times so that users on every platform are informed when a machine is idling rather than down. The machine Status Icon will show the dedicated icon for planned inactivity. Users of the Operator App and WEB application will see reasons automatically reported for any stops that fall within the scope of calendar events as soon as the stops begin.

Alerts

Act faster with Calls-to-Action and SMS notifications

Configure in-app calls-to-action or SMS notifications to alert key company personnel of urgent matters that require immediate attention. System administrators have extensive control to fine-tune their system's notifications to their particular company needs. Here are some examples of notifications that can be created:

  • If a particular machine is stopped or in setup mode over x amount of time
  • When a mold changeover or job completion is upcoming
  • Deviations in cycle time and quality alerts

Contact support@emerald.co.il for pricing. SMS package not included.

Multi-Step Production

A complete overview of a production journey in a single frame

The Product Tree provides an overview of the complete production process in a single frame so multi-step processes can be controlled and viewed with ease. Use the product tree as a quick navigation map or validate that your sync data was fully integrated.

The product tree is a new tab in the Product Form which shows upstream pathways of sub-components and materials. When a final product is viewed, it will show the complete production process as a flow chart broken up into of inputs and outputs for every intermediary process. The tree is collapsible and can be resized and re-centered for convenience. Learn more

Data Review

Perform calculations on your selected data directly within the App

Run your calculations directly on the data without needing to export it. Filter, sort and select items to refine your dataset, and then perform your calculations to calculate sums and averages, count the items on the list or find the MIN/MAX values. The calculations will respond dynamically to changes in the dataset - whether you adjust filters or change your selections by checking off different items.

Integrated documentation helps users master the system more quickly

Look for the buttons at the top-right corner to find step-by-step guides related to the page you are currently on. Articles in the Knowledge Base are updated on a continual basis, so make use of the Knowledge button added to the top bar to see more topics and view the complete table of contents. Improvements

Standardization

All tables now support view customizations, data sorting, filtering and calculations

Reports, forms and tables of every kind now support the same set of functions. If it appears as a table, regardless of the context, you can filter, sort, reorder columns, select items and perform calculations. The former distinction between reports and forms is now obsolete.

Table Defaults

Authorized users can modify report defaults for others

Authorized users can modify report defaults for others. Tailor the system's interface to the needs of your teams by creating a unified default that is group specific.

When report modifications are copied to others, they become the baseline default view. Individual users will still have the option to further personalize their reports but whenever they choose the option to Restore All Defaults, the modified report last copied to them will be restored.

Only users who have received permission to copy a report to others will see the button above relevant reports. Learn how to grant this permission to other users.

New Workflow

Check off multiple items to select or update them

Use the checkboxes in your tables to select items for various purposes:

  • Update forms
  • Export specific data
  • Run calculations on specific items

The new interface will now allow you to update multiple lines at once in any form. Effortlessly create orders, assign Inactivity Calendars, create configurations, or pre-approve molds for products. Learn more

Usability

Navigation presets limit the number of browser tabs

Internal links will open in a new browser tab by default. The system will open one browser tab per object type and will refresh the item on view with every new selection. Use your sidebar menu to open tabs in the view window or use the icon to open the page in a new browser tab. As long as your main page is refreshed and logged in, all other secondary tabs will remain logged in as well.

Configurations

Monitor Rejects Separately for Every Part in a Family Mold

Keep better inventory and improve costing by keeping accurate records for rejects and waste for processes that involve family molds. The Operator App now clearly indicates when a Configuration is in production and allows users to report rejects for any of the products (or parts) in production.

Personalization

Users can now change their own password

As a user, you can click on your username in the sidebar menu to change your own password. If you are a System Administrator, you can send a simple recovery password to another user without compromising security, and ask that user to reset their own password for themselves.
